# @immoassist/profil

Module Profil - Gestion du profil utilisateur et des paramètres.

## Installation

```bash
pnpm add @immoassist/profil
```

## Usage

```typescript
import { /* components */ } from '@immoassist/profil';
```

## Structure

- `types/` - Types TypeScript pour le module profil
- `components/` - Composants React spécifiques au profil
