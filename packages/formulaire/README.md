# @immoassist/formulaire

Module Formulaire - Gestion et création de formulaires.

## Installation

```bash
pnpm add @immoassist/formulaire
```

## Usage

```typescript
import { /* components */ } from '@immoassist/formulaire';
```

## Structure

- `types/` - Types TypeScript pour le module formulaire
- `components/` - Composants React spécifiques aux formulaires
