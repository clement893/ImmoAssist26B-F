# Base de connaissance OACIQ pour Léa

Ce document est chargé dans le contexte de Léa pour qu'elle connaisse les formulaires OACIQ et puisse accompagner les courtiers. **À compléter et maintenir** avec les descriptions officielles et bonnes pratiques.

---

## Rôle de l'OACIQ

L'**OACIQ** (Organisme d'autoréglementation du courtage immobilier du Québec) encadre la profession du courtage immobilier au Québec. Les formulaires OACIQ sont les documents officiels à utiliser dans les transactions immobilières pour assurer la conformité et la protection des parties.

---

## Catégories de formulaires

- **Obligatoire** : requis dans certaines situations (ex. promesse d'achat, contrat de courtage).
- **Recommandé** : bonnes pratiques, souvent utilisés pour sécuriser la transaction.
- **Curateur public** : formulaires spécifiques lorsque le curateur public est impliqué.

---

## Formulaires principaux (à compléter)

### PA – Promesse d'achat (Immeuble résidentiel)
- **Catégorie** : Obligatoire
- **Quand l'utiliser** : Pour formaliser l'offre d'achat d'un acheteur sur un immeuble résidentiel. À utiliser une fois que l'acheteur et le courtier ont convenu du prix et des conditions.
- **Infos clés** : Adresse du bien, nom des acheteurs et vendeurs, prix offert, date de clôture prévue, conditions (inspection, financement, etc.).
- **Conseil** : L'adresse du bien doit être complète (ville, province, code postal) avant de préparer le PA. Ne jamais assumer une transaction par défaut : demander « Pour quelle propriété ? » si l'utilisateur ne précise pas.

### DIA – Déclaration d'intention d'achat
- **Catégorie** : Selon le type (vente/achat)
- **Quand l'utiliser** : Document par lequel l'acheteur ou le vendeur déclare son intention et les modalités (ex. mandat, engagement). Souvent utilisé en début de mandat ou pour encadrer la relation courtier–client.
- **Conseil** : À distinguer de la promesse d'achat (PA) : la DIA peut précéder le PA dans le processus.

### CP – Contrat de courtage
- **Catégorie** : Obligatoire dans le cadre du mandat
- **Quand l'utiliser** : Pour établir la relation contractuelle entre le courtier et son client (vendeur ou acheteur). Précise le type de mandat, la durée, les conditions de rémunération.
- **Conseil** : À faire signer avant de commencer les démarches de vente ou d'achat.

### CCVE – Clause de cession – Vente d'entreprise
- **Catégorie** : Recommandé
- **Quand l'utiliser** : Lorsque la transaction implique la cession d'une entreprise en plus du bien immobilier (ex. fonds de commerce). Permet de formaliser les conditions de cession.
- **Conseil** : À proposer quand l'utilisateur mentionne une vente d'entreprise ou un fonds de commerce.

### Autres formulaires
Pour chaque autre formulaire OACIQ présent dans la plateforme (liste dans les « Données plateforme »), Léa s'appuie sur le **code**, le **nom** et l'**objectif** fournis par la base de données. Si l'utilisateur demande « quel formulaire pour X ? », proposer le formulaire dont l'objectif correspond le mieux et indiquer où le trouver : Transactions → ouvrir la transaction → onglet Formulaires OACIQ.

---

## Ordre logique typique (transaction résidentielle)

1. **Contrat de courtage (CP)** – établir le mandat
2. **Promesse d'achat (PA)** – une fois l'offre acceptée (achat)
3. **Autres formulaires** selon le cas (DIA, CCVE, etc.)

---

## Règles pour Léa

- Toujours indiquer le **code** du formulaire (ex. PA, DIA, CP) quand tu en parles.
- Si l'utilisateur demande « quels documents me manquent » ou « prochaine étape », se baser sur les formulaires listés dans les Données plateforme (brouillon / complété / signé) et proposer la suite logique.
- Ne jamais inventer un formulaire : n'indiquer que ceux présents dans le référentiel (Données plateforme ou ce document).
- Pour « préparer une promesse d'achat » ou « créer un PA », rappeler qu'il faut une transaction avec adresse complète, vendeurs et acheteurs identifiés ; confirmer après création que le formulaire a été créé et indiquer : Transactions → ouvrir la transaction → Formulaires OACIQ → compléter le PA.
